# Trend Data Tools

Tools for extracting Sweden-focused drug trend data from source spreadsheets and publishing it in researcher-friendly formats.

The `extractor/` package reads EUDA and FOHM workbooks and produces normalized dataset outputs in JSON and CSV. The `docs/` site renders that dataset as interactive charts and also provides direct dataset downloads for research use.

[Interactive Chart Viewer](https://martinsmeder.github.io/trend_data_tools/)
